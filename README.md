# KeenKit

## Entware usage

```
opkg update && opkg install curl && 
curl -L -s "https://raw.githubusercontent.com/spatiumstas/KeenKit/main/keenkit.sh" 
--output /opt/keenkit.sh && 
chmod +x /opt/keenkit.sh && 
/opt/keenkit.sh 
```
